kbsecret-discover-sessions
==========================

`kbsecret discover-sessions` scans your KBFS hierarchy for unconfigured KBSecret sessions and
helps the user configure them.

## Usage

```bash
$ kbsecret discover-sessions [--dry-run] [--auto-name]
```
