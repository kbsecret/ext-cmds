kbsecret-recovery
=================

`kbsecret-recovery` manages `recovery`-type records. The `recovery` record type is
available [here](https://github.com/kbsecret/ext-records/tree/master/recovery).

## Usage

```bash
$ kbsecret recovery [options] <load|next> <label>
```
