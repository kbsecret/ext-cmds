kbsecret-snip
=============

`kbsecret snip` uses [selecta](https://github.com/garybernhardt/selecta) to list all snippet
records in a session, and either prints or executes the selected one.

## Usage

```bash
$ kbsecret snip [--session <session>] [--execute]
```
